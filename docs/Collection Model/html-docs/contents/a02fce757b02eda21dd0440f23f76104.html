remove

UMLOperation
Collection :: Model :: util :: list :: List :: remove

Description

Removes all of the elements of this list that satisfy the given predicate. Errors or runtime exceptions thrown during iteration or by the predicate are relayed to the caller.

Parameters

Direction Name Type Description
in filter Predicate<E>

A predicate which returns 'true' for elements to be removed.
return successful boolean

Returns 'true' if the list was removed successfully, otherwise 'false'.
