addFirst

UMLOperation
Collection :: Model :: util :: list :: List :: addFirst

Description

Appends all of the elements in the specified collection at the beginning of this list, in the order that they are returned by the specified collection's iterator.

Parameters

Direction Name Type Description
in collection Collection<E>

The element to be appended to this list.
return successful boolean

Returns 'true' if the collection was added successfully, otherwise 'false'.
