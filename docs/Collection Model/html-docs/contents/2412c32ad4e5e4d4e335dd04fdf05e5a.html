remove

UMLOperation
Collection :: Model :: util :: list :: List :: remove

Description

Removes from this list all of its elements that are contained in the specified collection.

Parameters

Direction Name Type Description
in array E [1..*]

The array containing elements to be removed from this list.
return successful boolean

Returns 'true' if the collection elements was removed successfully, otherwise 'false'.
